1.How do you set the arguments that will be passed to the program when it's running?

- run [arglist]

2.How do you create a breakpoint?

- `b [file:]function` or `b [file:]lineNum`

3.How do you execute the next line of C code?

- n or s

4.How to dive into a function when debugging?

- s

5.How to continue the program after stopping at a breakpoint?

- c

6.How can you print the value of a variable in gdb?

- `p expr`

7.How do you configure gdb so it displays the value of a variable after every step?

- `display [/f] variable`

8.How to show a list of all variables and their values in the current function?

- `frame`, then `info locals`. 

9.How to quit out of gdb?

- quit
